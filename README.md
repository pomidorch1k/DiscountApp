# Обувная компания — Android приложение

Мобильное приложение для просмотра товаров обувной компании по заданию ГИА ДЭ.

## Карточка товара (по макету)

- **Левая секция** — фото товара (заглушка «Фото» при отсутствии)
- **Центральная секция** — Категория | Наименование, описание, производитель, поставщик, цена, единица измерения, количество на складе
- **Правая секция** — «Действующая скидка» с процентом

## Правила оформления

- **Скидка > 15%** — фон секции скидки **#2E8B57** (Sea Green)
- **Снижена цена** — исходная цена перечёркнута, красным, рядом итоговая цена чёрным
- **Нет на складе** — карточка выделена голубым цветом

## Требования

- Android Studio Hedgehog (2023.1.1) или новее
- Android SDK 34
- Min SDK 24

## Сборка и запуск

1. Откройте проект в **Android Studio** (File → Open → выберите папку `DiscountApp`)
2. Дождитесь синхронизации Gradle (Android Studio автоматически создаст wrapper при необходимости)
3. Подключите устройство или запустите эмулятор
4. Нажмите **Run** (▶️) или `Shift+F10`

### Сборка из командной строки

Если Gradle Wrapper уже настроен:

```
cd DiscountApp
gradlew assembleDebug
```

APK будет в: `app/build/outputs/apk/debug/app-debug.apk`

## Что реализовано сейчас

- **Покупка товара**: на карточке есть кнопка «Купить». При покупке количество на складе уменьшается, а при нуле товара карточка подсвечивается голубым и купить уже нельзя.
- **Фильтр по названию**: над списком есть строка поиска — вводите часть названия, и список фильтруется по имени товара.
- **Админ‑панель**: в панели действий (иконка шестерёнки) открывается экран администратора, где можно:
  - выбрать существующий товар и отредактировать его,
  - добавить новый товар, заполнив форму.

## APK‑гайд (как получить и установить)

1. Соберите APK (debug или release) через Android Studio или GitHub Actions.
2. Найдите файл:
   - debug: `app/build/outputs/apk/debug/app-debug.apk`
   - release: `app/build/outputs/apk/release/app-release.apk`
3. Скопируйте APK на телефон (USB/облако/мессенджер).
4. На телефоне откройте APK и подтвердите установку (при необходимости разрешите установку из неизвестных источников).
